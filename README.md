# ICD-FPTU: Benchmarking Image Capture Duplication Rate and F1-Score for Pothole Surveillance Systems

This repository contains the dataset, experimental results, and method implementations for the paper:

**"ICD-FPTU: Benchmarking Image Capture Duplication Rate and F1-Score for Pothole Surveillance Systems"**

---

## 📂 Repository Structure

- **dataset/**  
  Contains the full dataset used in the experiments. This dataset is publicly available for benchmarking filtering module for pothole surveillance systems.

- **detect/**  
  Includes detailed results and outputs from different detection methods applied in the study. Each subfolder corresponds to a specific method.

- **final_results.xlsx**  
  A summary file consolidating the performance metrics across all methods.

---

## 📊 Benchmarking Metrics

The study evaluates pothole surveillance systems using two key metrics:

- **Image Capture Duplication Rate (ICD):**  
  Measures redundancy in captured images, and detected potholes to assess efficiency.

- **F1-Score:**  
  Evaluates detection accuracy by balancing precision and recall.

---

## 🚀 Purpose

This benchmark provides:
- A standardized dataset for Image Capture Duplication research.
- Comparative results across multiple methods.
- A foundation for future improvements in intelligent transportation and road maintenance systems.
